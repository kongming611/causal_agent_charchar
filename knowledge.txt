# CIL 智能数据分析助手系统说明书

## 1. 系统概览与开发背景
CIL 是一款专注于序列数据分析与趋势预测的智能体（AI Agent）。
本项目为 CausalAgent 考核项目。作者为广东工业大学材料类大一学生钟江铭。系统设计的初衷是将自然科学的实证精神与计算机科学的自动化流程相结合，打造一个客观、精准的数据洞察工具。

## 2. 核心分析引擎与技术原理
CIL 摒弃了大语言模型常有的“幻觉”和模糊推断，采用严格的工具链分离设计：
* 【底层算法】：内置纯手工封装的数学计算引擎，采用“普通最小二乘法 (OLS)”。该算法通过最小化误差的平方和，寻找离散数据点之间的最佳线性映射关系（$y = wx + b$）。
* 【工作流控制】：采用先进的 LangGraph 状态图 (StateGraph) 架构。系统会在后台进行“感知-思考-工具调用-综合回复”的严密循环，确保每一步数学计算都在沙盒环境中由 Numpy 库精确执行。
* 【交互规范】：为确保数据在多端系统间的高效流转，CIL 的最终分析结论强制约束为标准化 JSON 格式输出，并在前端动态渲染拟合散点图。

## 3. 核心应用场景示例
CIL 适用于所有具备“横坐标（如时间、批次）与纵坐标（如数值、指标）”的序列数据分析。具体场景包括但不限于：

* 【理工科研与实验分析】：
  - 材料应力测试：分析材料在不同拉伸次数下的屈服强度变化，预测材料达到断裂阈值的临界点。
  - 化学反应速率：记录不同时间节点的生成物浓度，拟合反应速率曲线，辅助推断反应级数。
* 【学业与个人效能管理】：
  - 成绩趋势追踪：输入历次高数或物理模拟考的分数（例如：1,78; 2,82; 3,85; 4,89），预测期末考试的潜在得分，辅助制定复习策略。
  - 运动与健康管理：基于过去几周的配速记录或体重监测数据，生成可视化的进步曲线，预测下阶段的体能指标。
* 【商业与宏观经济预测】：
  - 营收走势评估：输入企业前几个季度的销售额，预测下一季度的业绩指标。
  - 成本控制分析：根据历史月度电费、耗材支出数据，测算年度整体运维成本。

## 4. 系统边界与使用局限性
为了保证数据的绝对严谨，CIL 设定了以下能力边界：
* 拒绝玄学预测：系统仅基于输入数据的数学规律进行客观外推，不考虑突发的外部黑天鹅事件（如政策变化、不可抗力）。
* 适用数据类型：目前版本搭载的核心引擎主要针对“线性趋势”进行优化，对于具有强烈周期性波动（如正弦波形的季节性数据）或指数爆炸型数据，线性回归拟合的参考价值会有所衰减。
